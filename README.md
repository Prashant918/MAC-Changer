> Linux and Mac OSX.

addresses of network interfaces.

## Uses of MAC Changer
``` Command
pip3 install -r requirement.txt
python3 mac_changer.py 
```
If you want a help... you can easily type -h 

```
python3 mac_changer.py -h
```
